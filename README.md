fxpoc
=====
JavaFx proof-of-concept app

an application based on JavaFx that should help to evaluate if it is
feasible to use JavaFX as the next UI framework for CompendiumNG

prerequisites
=============
java JDK 1.7.0 u51
gradle (optional since gradlewrapper is included)


how to build
============
git clone ....
cd project_work_dir
gradle build


how to run it
=============


linux & Mac
===========

java -jar ./build/fxpoc-...jar
or 
./gradlew run


Windows
=======

java -jar .\build\fxpoc-...jar
or
gradlew.bat run
